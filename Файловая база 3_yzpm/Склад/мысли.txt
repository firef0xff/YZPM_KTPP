площадь произвольного многоугольника

Я два способоа знаю: по трапециям и по треугольникам. Второй проще:
находишь ориентированные площади треугольников (через определитель), две вершины, которого - это вершины стороны многоугольника, а третья начало координат. Сумма этих площадей и будет ответ. Время O(N), то есть количество вершин ограничего фак тически доступной памятью, хотя можно и не хранить все координаты вершин Алгоритм работает для любого просто многоугольника, те без самоналожений и самопересечений. 

Ориентированная площадь треугольника


Дан треугольник
Необходимо найти его ориентированную площадь
Ориентированная площадь треугольника находится через определитель:

     |x1, y1, 1|
S2 = |x2, y2, 1| = x1y2 + x2y3 + x3y1 - y1x2 - y2x3 - y3x1
     |x3, y3, 1|

S = 1/2 * S2

double area_triangle (point a, point b, point c)
{
    return 0.5 * (a.x * b.y + b.x * c.y + c.x * a.y - a.y * b.x - b.y * c.x - c.y * a.x);
}
